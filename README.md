# Email Writing Simulator

This is a simple, web-based simulator for **primary school students** to practise writing emails in a safe environment.

This tool **does not send real emails**, **requires no sign-in**, and **stores no data**.

---

## For Teachers

- Designed to teach **email structure** (To, Subject, Message)
- Suitable for **English, ICT, and Cyber Wellness lessons**
- Uses **2 Stars and 1 Wish** for simple formative feedback
- Highlights common mistakes (e.g. missing `@` in email address)
- Safe for classroom use
